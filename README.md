📊 Crypto Dashboard

Aplicação web desenvolvida em JavaScript e React que consome uma API de criptomoedas para exibir informações atualizadas sobre preços, variações e dados de mercado.
O objetivo do projeto é praticar consumo de APIs, manipulação de dados e criação de interfaces modernas com React.

🚀 Tecnologias utilizadas

JavaScript
React
HTML5
CSS3
API de criptomoedas

📦 Funcionalidades

📈 Visualizar preço atual das criptomoedas
🔄 Atualização de dados em tempo real através da API
📊 Exibição de informações como:
Nome da moeda
Preço atual
Variação de mercado
Ranking

🎯 Objetivo do projeto

Este projeto foi desenvolvido para estudo e prática de:
Consumo de APIs REST
Desenvolvimento de interfaces com React
Manipulação de dados em JavaScript
Organização de projetos front-end

📚 Aprendizado

Durante o desenvolvimento foram praticados conceitos como:
Hooks do React
Requisições HTTP
Componentização
Manipulação de estado
